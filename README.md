# IOT_09_ESP-Lan
Atividade desenvolvida com ESP32 e DHT, configurando o ESP32 como servidor e controle do led, enquanto outra placa recebe e realiza o envio da leitura do sensor. Desenvolvido em aula da de IOT do curso Desenvolvimento de Sistemas do Senai

Segue um vídeo do código sendo executado em circuito:
https://www.youtube.com/watch?v=IMc014zxy-M

## Tecnologias utilizadas
* C++
* Arduino IDE
* Wokwi
* ESP32
* DHT
